# CPP Module 09 - STL Containers

This project is the final module in the C++ series at 42. It focuses on the practical application of the **Standard Template Library (STL)**, emphasizing container selection, algorithm efficiency, and the constraints of the C++98 standard.

## 🚀 Projects Overview

### Ex00: Bitcoin Exchange
A program that outputs the value of a certain amount of bitcoin on a certain date.
- **Key Concepts:** `std::map`, file parsing (CSV), date validation.
- **Goal:** Efficiently handle a large database and perform lookups.

### Ex01: Reverse Polish Notation (RPN)
A mathematical expression evaluator using Reverse Polish Notation.
- **Key Concepts:** `std::stack`, expression parsing, error handling.
- **Goal:** Processing tokens and maintaining operation precedence.

### Ex02: PmergeMe
An implementation of the **Ford-Johnson algorithm** (merge-insertion sort) to sort large sequences of integers.
- **Key Concepts:** Template programming, container comparison (`std::vector` vs `std::deque`), time complexity analysis.
- **Goal:** Sorting data while minimizing comparisons and comparing the performance of different STL containers.

## 🛠️ Requirements
- Compiles with `c++`
- Flags: `-Wall -Wextra -Werror -std=c++98`
